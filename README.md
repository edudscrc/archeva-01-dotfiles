# ArchEVA-01
[Arch Linux + Hyprland] Dotfiles for Hyprland.
Theme with colors inspired by EVA-01 from the anime Neon Genesis Evangelion.

## Installation
<pre>
  $ git clone https://github.com/edudscrc/archeva-01-dotfiles.git
  $ cd archeva-01-dotfiles
  $ chmod +x ./install.sh
  $ ./install.sh
</pre>
